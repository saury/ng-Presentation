FLEX

主角登场: FLEX


Flex是Flexible Box的缩写，意为"弹性布局"，用来为盒状模型提供最大的灵活性。


Flex布局模块旨在提供一个更佳有效的布局方式，更好的控制项目的对齐和自由分配容器空间，即使它们的大小是未知的或动态的。因此得其名"flex"。

兼容性

目前，它已经得到了除IE10以下版本的主流浏览器的支持，这意味着，可以较放心地使用这项功能。

flex属性

常规用法： 

                
            .item {
                flex: [number];
            }

实际上这个属性有些复杂

                
            .item {
                flex: <'flex-grow 扩展比'>|<'flex-shrink 收缩比'>|<'flex-basis' 伸缩基准值>
            }
